=== HOW TO RUN ===

Backend:
1. cd backend
2. npm install
3. npm start (runs on port 3000)

Frontend:
1. Open new terminal
2. python -m http.server 8000
3. Visit http://localhost:8000

=== API STRUCTURE ===

Request: POST /api/generate
{
  "topic": "Photosynthesis"
}

Response:
{
  "topic": "...",
  "overview": "...",
  "questions": [
    {
      "id": 1,
      "question": "...",
      "options": ["...", "...", "...", "..."],
      "correctIndex": 0,
      "explanation": "..."
    }
  ],
  "explanation": "...",
  "challenge": {
    "prompt": "...",
    "placeholder": "...",
    "acceptableAnswers": ["...", "..."],
    "explanation": "..."
  }
}

=== DESIGN DECISIONS ===

1. Vanilla JavaScript (No frameworks)
   - Faster initial load
   - Minimal dependencies
   - Direct DOM control

2. Inline CSS
   - Single file deployment
   - No build step needed
   - Consistent styling

3. Backend-Only Architecture
   - Secure API key management
   - CORS for cross-origin requests
   - Stateless API design

4. Progressive Enhancement
   - Works without JavaScript (graceful fallback planned)
   - Responsive on all devices
   - Accessible components

5. Clean Light Theme
   - Blue (#0066ff) as primary accent
   - White/slate background
   - Soft shadows for depth
   - Clear hierarchy
